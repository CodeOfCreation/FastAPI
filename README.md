# FastAPI Practice & Projects

This repository contains my learning journey with **FastAPI**, covering core concepts, hands-on practice, and real-world projects. It includes examples of building REST APIs, request handling, data validation, routing, dependency injection, database integration, authentication, and deployment basics.

## What's Inside

* FastAPI fundamentals and practice examples
* CRUD API development
* Request and response validation using Pydantic
* Path, query, and body parameters
* Database integration
* Authentication and authorization examples
* Project-based implementations
* API testing and interactive documentation
* Best practices for structuring FastAPI applications

## Tech Stack

* Python
* FastAPI
* Pydantic
* Uvicorn
* SQLAlchemy (where applicable)
* SQLite/PostgreSQL (project dependent)

This repository is intended for learning, experimentation, and building a strong foundation in FastAPI by progressing from basic concepts to complete API projects.
